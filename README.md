# E-Commerce Store Web Application

## Live Demo
[🛒 View Live](https://krishshna.github.io/ecommerce-store-js/)

## Features
- Loads products dynamically from Fake Store API
- Product search (live filtering)
- Category filter
- Add to cart with LocalStorage persistence
- Cart page with images and total price

## Tools & Technologies
- HTML, CSS, JavaScript
- Fetch API, DOM manipulation, LocalStorage
